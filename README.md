## 🎮 About

**Block Fusion** is a high-stakes, grid-based logic puzzle. Strategically merge identical blocks to evolve them into higher-tier elements while managing limited board space.

---

## 🕹️ Mechanics

| Mechanic | Description |
|----------|-------------|
| 🔀 **Fusion** | Drag or slide blocks to combine matching values |
| ⚡ **Cascades** | Clear space by triggering automatic chain reactions |
| 🧠 **Strategy** | Every move adds new blocks; plan ahead to avoid a deadlock |

---

## 🛠️ Technical Overview

| Component | Detail |
|-----------|--------|
| 🔄 **Logic** | Recursive merging algorithm |
| 💾 **State** | JSON-based save/load system |
| 🎨 **UI** | Responsive CSS Grid / Canvas rendering |
